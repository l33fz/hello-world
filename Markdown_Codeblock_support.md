1. **Mermaid Diagrams**  
   ```mermaid
   graph TD;
       A-->B;
       A-->C;
       B-->D;
       C-->D;
       D-->A
   ```

2. **PlantUML Diagrams**  
   ```plantuml
   @startuml
   Alice -> Bob: Hello
   Bob -> Alice: Hi!
   @enduml
   ```

3. **LaTeX for Mathematical Expressions**  
   ```latex
   E = mc^2
   ```

4. **Graphviz for Graphs**  
   ```dot
   digraph G {
       A -> B;
       B -> C;
       C -> A;
   }
   ```

5. **Asciidoc for AsciiDoc Content**  
   ```asciidoc
   = Document Title
   Author Name <author@example.com>
   :doctype: book

   == Section Title
   This is a section in AsciiDoc.
   ```

6. **CSV for Comma-Separated Values**  
   ```csv
   Name, Age, City
   Alice, 30, New York
   Bob, 25, Los Angeles
   ```

7. **TOML for Tom's Obvious, Minimal Language**  
   ```toml
   title = "TOML Example"
   [owner]
   name = "Tom Preston-Werner"
   dob = 1979-05-27T07:32:00Z
   ```

8. **INI for Initialization Files**  
   ```ini
   [database]
   user = admin
   password = secret
   host = localhost
   ```

9. **Shell Commands**  
   ```bash
   # List files in the current directory
   ls -la
   ```

10. **R Markdown for R Code**  
    ```r
    # Simple R code to calculate the mean
    data <- c(1, 2, 3, 4, 5)
    mean(data)
    ```
